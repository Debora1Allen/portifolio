# 👽 Dead Alien - Portfólio & Arquivo Literário

> "Registros, contos e anomalias documentadas."

Um projeto front-end que une desenvolvimento de software e escrita criativa em uma experiência imersiva. Construído com estética _brutalist_, efeitos _glitch_ e design centrado em atmosfera de horror cósmico e sci-fi dark.

---

## 🛠️ Tecnologias Utilizadas

- **HTML5:** Estruturação semântica.
- **CSS3:** Variáveis customizadas, Grid/Flexbox, animações _keyframes_ e tipografia _monospace_.
- **JavaScript (Vanilla):** Manipulação de DOM, sistema de filtros dinâmicos, reprodução de áudio e integração com API de conteúdo.
- **Arquitetura Headless:** Consumo de textos e mídias via API externa.

---

## ⚙️ Funcionalidades

- **Interface Imersiva:** Design focado na experiência de "acessar arquivos confidenciais".
- **Filtro de Categorias:** Sistema rápido de separação de relatórios (Horror Cósmico, Slasher, Sci-Fi).
- **Terminal de Leitura (Modal):** Os textos abrem sobrepostos à tela inicial, travando a rolagem de fundo e simulando um terminal de sistema.
- **Integração de Áudio:** Trilha sonora original para cada conto, com controles de interface customizados `[ ÁUDIO: LIGADO ]`.

---

## 🚀 Como Executar o Projeto Localmente

1. Clone este repositório:
   \`\`\`bash
   git clone https://github.com/Debora1Allen/portifolio
   \`\`\`
2. Navegue até a pasta do projeto.
3. Abra o arquivo \`index.html\` no seu navegador ou utilize a extensão _Live Server_ do VS Code.

_(Nota: Como o conteúdo literário e as mídias são consumidos de uma API externa privada, o projeto local fará a requisição a esses arquivos remotamente)._
